The `Database` class represents a jasonisnthappy database instance.

## Opening a database

### open

Opens a database at the specified path.

```python
@staticmethod
def open(path: str) -> Database
```

<ParamField path="path" type="str" required>
  Path to the database file
</ParamField>

<ResponseField name="Database" type="Database">
  Database instance
</ResponseField>

**Example:**

```python
from jasonisnthappy import Database

db = Database.open("./my_database.db")
try:
    # Use database
    pass
finally:
    db.close()
```

### open_with_options

Opens a database with custom options.

```python
@staticmethod
def open_with_options(path: str, options: CDatabaseOptions) -> Database
```

<ParamField path="path" type="str" required>
  Path to the database file
</ParamField>

<ParamField path="options" type="CDatabaseOptions" required>
  Custom database configuration options
</ParamField>

<ResponseField name="Database" type="Database">
  Database instance
</ResponseField>

**Example:**

```python
opts = Database.default_database_options()
opts.read_only = True
opts.cache_size = 2000

db = Database.open_with_options("./my_database.db", opts)
```

### default_database_options

Returns the default database options.

```python
@staticmethod
def default_database_options() -> CDatabaseOptions
```

<ResponseField name="CDatabaseOptions" type="CDatabaseOptions">
  Default database configuration
</ResponseField>

## Database management

### close

Closes the database and frees associated resources.

```python
def close() -> None
```

**Example:**

```python
db.close()
```

### Context manager support

Use the `with` statement for automatic resource cleanup.

**Example:**

```python
with Database.open("./my_database.db") as db:
    # Database is automatically closed when exiting the block
    tx = db.begin_transaction()
    # ...
```

### begin_transaction

Begins a new transaction.

```python
def begin_transaction() -> Transaction
```

<ResponseField name="Transaction" type="Transaction">
  Transaction instance
</ResponseField>

**Example:**

```python
tx = db.begin_transaction()
try:
    # Perform operations
    tx.commit()
except Exception:
    tx.rollback()
    raise
```

### get_collection

Gets a collection reference for non-transactional operations.

```python
def get_collection(name: str) -> Collection
```

<ParamField path="name" type="str" required>
  Name of the collection
</ParamField>

<ResponseField name="Collection" type="Collection">
  Collection handle
</ResponseField>

**Example:**

```python
coll = db.get_collection("users")
try:
    doc_id = coll.insert({"name": "Alice", "age": 30})
finally:
    coll.close()
```

## Index management

```python

```

<ParamField path="collection_name" type="str" required>

</ParamField>

<ParamField path="index_name" type="str" required>
  Name for the index
</ParamField>

<ParamField path="field" type="str" required>

</ParamField>

<ParamField path="unique" type="bool" default="False">

## Schema validation


<ParamField path="collection_name" type="str" required>
  Name of the collection
</ParamField>

<ParamField path="schema" type="Dict[str, Any]" required>

</ParamField>

**Example:**

```python

        "age": {"type": "number"},

    "required": ["name", "email"],
```

## Web UI

### start_web_ui

Starts the web UI server at the given address.

```python
def start_web_ui(addr: str) -> WebServer
```

<ParamField path="addr" type="str" required>
  Address to bind the web server to (e.g., "127.0.0.1:8080")
</ParamField>

<ResponseField name="WebServer" type="WebServer">
  WebServer handle to control the server
</ResponseField>

**Example:**

```python
server = db.start_web_ui("127.0.0.1:8080")
print("Web UI available at http://127.0.0.1:8080")
# ... later ...
server.stop()
```
