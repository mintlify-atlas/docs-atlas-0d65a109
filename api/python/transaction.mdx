The `Transaction` class represents a database transaction.

## Transaction lifecycle

### commit

Commits the transaction.

```python
def commit() -> None
```

**Example:**

```python
tx = db.begin_transaction()
try:
    # Perform operations
    tx.commit()
except Exception:
    tx.rollback()
    raise
```

### rollback

Rolls back the transaction.

```python
def rollback() -> None
```

**Example:**

```python
tx = db.begin_transaction()
try:
    # Perform operations
    tx.commit()
except Exception:
    tx.rollback()
    raise
```

### Context manager support

Transactions automatically commit on success and rollback on exception.

**Example:**

```python
with db.begin_transaction() as tx:
    doc_id = tx.insert("users", {"name": "Alice", "age": 30})
    # Transaction is automatically committed if no exception occurs
    # or rolled back if an exception is raised
```

### is_active

Checks if the transaction is still active.

```python
def is_active() -> bool
```

<ResponseField name="bool" type="bool">
  True if the transaction is active
</ResponseField>

## Document operations

### insert

Inserts a document into a collection.

```python
def insert(collection_name: str, doc: Dict[str, Any]) -> str
```

<ParamField path="collection_name" type="str" required>
  Name of the collection
</ParamField>

<ParamField path="doc" type="Dict[str, Any]" required>
  Document to insert
</ParamField>

<ResponseField name="str" type="str">
  ID of the inserted document
</ResponseField>

**Example:**

```python
tx = db.begin_transaction()
try:
    doc_id = tx.insert("users", {
        "name": "Alice",
        "age": 30,
        "email": "alice@example.com"
    })
    print(f"Inserted document with ID: {doc_id}")
    tx.commit()
except Exception:
    tx.rollback()
    raise
```

### find_by_id

Finds a document by its ID.

```python
def find_by_id(collection_name: str, doc_id: str) -> Optional[Dict[str, Any]]
```

<ParamField path="collection_name" type="str" required>
  Name of the collection
</ParamField>

<ParamField path="doc_id" type="str" required>
  Document ID to find
</ParamField>

<ResponseField name="Optional[Dict[str, Any]]" type="Optional[Dict[str, Any]]">
  Document if found, None otherwise
</ResponseField>

**Example:**

```python
doc = tx.find_by_id("users", "user123")
if doc:
    print(f"Found: {doc}")
else:
    print("Document not found")
```
