The `Collection` class represents a non-transactional collection handle.

## Collection management

### close

Frees the collection handle.

```python
def close() -> None
```

**Example:**

```python
coll = db.get_collection("users")
try:
    # Use collection
    pass
finally:
    coll.close()
```

### Context manager support

Use the `with` statement for automatic resource cleanup.

**Example:**

```python
with db.get_collection("users") as coll:
    doc_id = coll.insert({"name": "Bob", "age": 25})
    # Collection is automatically closed when exiting the block
```

### name

Gets the collection name.

```python
def name() -> str
```

<ResponseField name="str" type="str">
  Collection name
</ResponseField>

## Query and filter operations

### find

Finds documents matching a filter.

```python
def find(filter_str: str) -> List[Dict[str, Any]]
```

<ParamField path="filter_str" type="str" required>
  Query filter expression
</ParamField>

<ResponseField name="List[Dict[str, Any]]" type="List[Dict[str, Any]]">
  List of matching documents
</ResponseField>

**Example:**

```python
results = coll.find("age > 25")
for user in results:
    print(user)
```

### find_one

Finds first document matching a filter.

```python
def find_one(filter_str: str) -> Optional[Dict[str, Any]]
```

<ParamField path="filter_str" type="str" required>

</ParamField>

<ResponseField name="Optional[Dict[str, Any]]" type="Optional[Dict[str, Any]]">

</ResponseField>

**Example:**


## Upsert operations

### upsert_by_id

Upserts a document by ID. Returns UpsertResult with id and inserted flag.

```python
def upsert_by_id(doc_id: str, doc: Dict[str, Any]) -> UpsertResult
```

<ParamField path="doc_id" type="str" required>
  Document ID
</ParamField>

<ParamField path="doc" type="Dict[str, Any]" required>
  Document data
</ParamField>

<ResponseField name="UpsertResult" type="UpsertResult">
  Result containing id and whether it was inserted or updated
</ResponseField>

**Example:**

```python
result = coll.upsert_by_id("user123", {"name": "Alice", "age": 30})
if result.inserted:
    print(f"Inserted new document: {result.id}")
else:
    print(f"Updated existing document: {result.id}")
```

### upsert

Upserts a document matching a filter. Returns UpsertResult with id and inserted flag.

```python
def upsert(filter_str: str, doc: Dict[str, Any]) -> UpsertResult
```

<ParamField path="filter_str" type="str" required>
  Query filter expression
</ParamField>

<ParamField path="doc" type="Dict[str, Any]" required>
  Document data
</ParamField>

<ResponseField name="UpsertResult" type="UpsertResult">
  Result containing id and whether it was inserted or updated
</ResponseField>

**Example:**

```python
result = coll.upsert("email == 'alice@example.com'", {
    "name": "Alice",
    "email": "alice@example.com",
    "age": 30
})
```

## Bulk operations

### insert_many

Inserts multiple documents. Returns list of IDs.

```python
def insert_many(docs: List[Dict[str, Any]]) -> List[str]
```

<ParamField path="docs" type="List[Dict[str, Any]]" required>
  List of documents to insert
</ParamField>

<ResponseField name="List[str]" type="List[str]">
  List of inserted document IDs
</ResponseField>

**Example:**

```python
ids = coll.insert_many([
    {"name": "Alice", "age": 30},
    {"name": "Bob", "age": 25},
    {"name": "Charlie", "age": 35}
])
print(f"Inserted {len(ids)} documents")
```

### bulk_write

Executes multiple operations in a transaction.

```python
def bulk_write(operations: List[Dict[str, Any]], ordered: bool = True) -> Dict[str, Any]
```

<ParamField path="operations" type="List[Dict[str, Any]]" required>
  List of bulk operations
</ParamField>

<ParamField path="ordered" type="bool" default="True">
  Whether to execute operations in order
</ParamField>

<ResponseField name="Dict[str, Any]" type="Dict[str, Any]">
  Result with statistics for each operation type
</ResponseField>

**Example:**

```python
operations = [
    {"op": "insert", "doc": {"name": "Alice", "age": 30}},
    {"op": "update", "filter": "age > 25", "update": {"verified": True}},
    {"op": "delete", "filter": "age < 18"}
]
result = coll.bulk_write(operations)
print(f"Bulk write result: {result}")
```

### aggregate

Executes an aggregation pipeline.

```python
def aggregate(pipeline: List[Dict[str, Any]]) -> List[Dict[str, Any]]
```

<ParamField path="pipeline" type="List[Dict[str, Any]]" required>
  Aggregation pipeline stages
</ParamField>

</ResponseField>

**Example:**

```python

]
results = coll.aggregate(pipeline)
for result in results:
    print(result)
```

## Query builder helpers

### query_with_options

Executes a query with all options.

```python
def query_with_options(
    filter_str: Optional[str] = None,
    sort_field: Optional[str] = None,
    sort_asc: bool = True,
    limit: int = 0,
    skip: int = 0,
    project_fields: Optional[List[str]] = None,
    exclude_fields: Optional[List[str]] = None,
) -> List[Dict[str, Any]]
```

<ParamField path="filter_str" type="Optional[str]">
  Query filter expression
</ParamField>

<ParamField path="sort_field" type="Optional[str]">
  Field to sort by
</ParamField>

<ParamField path="sort_asc" type="bool" default="True">
  Sort in ascending order
</ParamField>

<ParamField path="limit" type="int" default="0">
  Maximum number of results (0 = no limit)
</ParamField>

<ParamField path="skip" type="int" default="0">
  Number of results to skip
</ParamField>

<ParamField path="project_fields" type="Optional[List[str]]">
  Fields to include in results
</ParamField>

<ParamField path="exclude_fields" type="Optional[List[str]]">
  Fields to exclude from results
</ParamField>

<ResponseField name="List[Dict[str, Any]]" type="List[Dict[str, Any]]">
  List of matching documents
</ResponseField>

**Example:**

```python
results = coll.query_with_options(
    filter_str="age > 25",
    sort_field="age",
    sort_asc=False,
    limit=10,
    skip=5,
    project_fields=["name", "age"]
)
```

## Watch and change streams

### watch

Starts watching for changes on the collection.

```python
def watch(
    callback: Callable[[str, str, Optional[Dict[str, Any]]], None],
    filter_str: Optional[str] = None,
) -> WatchHandle
```

<ParamField path="callback" type="Callable[[str, str, Optional[Dict[str, Any]]], None]" required>
  Callback function receiving (operation, doc_id, document)
</ParamField>

<ParamField path="filter_str" type="Optional[str]">
  Optional filter for watched events
</ParamField>

<ResponseField name="WatchHandle" type="WatchHandle">
  Handle to stop the watch
</ResponseField>

**Example:**

```python
def on_change(op, doc_id, doc):
    print(f"{op}: {doc_id}")
    if doc:
        print(f"  Data: {doc}")

handle = coll.watch(on_change)
# ... later ...
handle.stop()
```
